自增的问题探究

1.自增


概念：在编程中，我们可以写出i = i + 1 以及 i += 1，还有++i,i++
这类基本都是调用了自身的变量并且使其再加1

为什么提倡写成i++，++i的形式？


写成i++，++i的形式最主要的原因是c语言可以直接转换为汇编语言，在n年前的cpu汇编指令就有++的指令，所以c也有

这样写有什么好处？


1.可以使代码更精炼
2.可以使自增速度更快

前自增与后自增的区别和相同：

相同：执行结果都令i的值+1

不同：


i++ 先运算再自增
++i 先自增再运算

我们可以用c语言一段代码来观察双方的差距

/*自增/自减的问题探究

时间：2023.7.7日下午18.19分

*/

#include

int main()

{

int i;

int j;

int k;

int m;

i = j = 3; //等价于i = 3， j = 3;

printf("i = %d j = %d k = %d m = %d\n",i,j,k,m);
