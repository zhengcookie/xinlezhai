进制的问题

1.n进制


概念：逢n进一
比如十进制：就是到10这个数字，进一位.
十六进制：由0-9，A-F组成，字母不区分大小写。
其他的也是同理。

2.把n进制转为十进制（乘以n，每次都为他的他的n次方）


比如五进制转换十进制：234
就这样转化 ：4+3x5+2x5^2
比如十六进制转换十进制：234C
就这样转化 ：12+4x16+3x16^2+2x16^3

3.十进制转为n进制(除以n不断取它的余数)


比如十进制转化为二进制：185
那么就这样计算：185 % 2 = 1 .. 92 % 2 = 0 46 % 2 = 0 .. 1 % 2 = 1 结果为10111001(倒着写)
比如八进制转化为十进制：185
那么就这么计算：185 % 8 = 1 23 % 8 = 7 2 % 8 = 2 结果为271(倒序)

4.不同进制所代表的数值之间的关系

十进制的3981转化为十六进制为 F8D 他们所代表的是同一类关系！

代码演示如下

#include int main() { int i; i = 0xF8D; printf("%d\n",i); { printf("%X\n",i); } }
